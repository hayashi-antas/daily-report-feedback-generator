# Google Sheets 日報フィードバック生成（研修成果物）

## 概要
本リポジトリは、Google Sheets 上で運用している日報に対し、
生成AI（Gemini / Vertex AI）を用いてフィードバックコメントを生成する仕組みを、
研修の一環として設計・検討した内容をまとめたものです。

## 背景・目的
- 日報運用の負担軽減
- フィードバックの質・一貫性の向上
- 生成AIを業務に組み込む際の設計・仕様策定の練習

## 成果物
- 仕様書（SPEC.md）
- プロンプト仕様（SPEC.md 内）
- Google Apps Script の実装例（GAS.md）

## 特徴
- 仕様駆動でプロンプトを設計
- Vertex AI を利用した企業向け運用を想定
- 手動実行（ボタン押下）による安全な生成

## 注意事項
- 本リポジトリ内のコードは研修用サンプルです
- 実運用時は権限設定や機密情報の管理に十分注意してください
